# SimpleOps website

Static website for [simpleops.co.nz](https://simpleops.co.nz).

## Hosting

Designed for Cloudflare Pages or Netlify. The contact form currently includes
Netlify Forms attributes; use a compatible form endpoint when deploying
elsewhere.

## Before public launch

- Replace the temporary founder wording on `about.html`.
- Confirm `hello@simpleops.co.nz` is active.
- Have the privacy policy, website terms and paid-service agreements reviewed.
- Confirm Auckland travel and service-area wording.
- Test contact-form notifications after deployment.
